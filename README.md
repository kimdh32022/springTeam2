"반려동물과 함께, 더 따뜻한 연결! 커뮤니티부터 병원 검색, 중고거래까지 한 곳에서!"


## 기술 스택
### Front-end
- 템플릿 엔진 : Thymeleaf
- 언어 : HTML, JavaScript
### Back-end
- 프레임워크: SpringBoot
- DBMS : MariaDB
- 언어 : Java(17)
### Tools
- IDE : IntelliJ
- Version Controll : Git, Github
